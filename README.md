# REACT TRAINING

GitHelp:
-------

* Add files for commit:
    git add *

* Commit changes:
    git commit -m "commit message"

* Add files and commit:
    git commit -a -m "commit message"

* Undo changes:
    git checkout -- <Filename.xx>

* Undo commit:
    git reset HEAD~

* Undo commit without send it to staged:
    git reset --soft HEAD~
